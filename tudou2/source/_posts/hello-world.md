# 添加标签页
前面通过修改next主题下的_config.yml文件中的menu选项，可以在主页面的菜单栏添加标签选项，但是此时点击标签，跳转的页面会显示page not found。
添加标签页面的具体方法是：
``` bash
$hexo new page tags
```

# 添加文章
打開 Git Bash，進入blog 所在文件夾
``` bash
$hexo new [时间处理] #新建名為`時間處理`的文章
INFO  Created: D:\Program Files\blog\source\_posts\时间处理.md #反饋信息文章目錄
```
